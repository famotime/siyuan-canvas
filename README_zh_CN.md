# 思源 Canvas

在思源笔记中打开、编辑和导出 Obsidian `.canvas` 画布文件。基于开放的 JSON Canvas 格式，实现两个笔记工具之间的画布互通。

## 它能做什么

### 画布文件管理

- 新建空白画布，或从本地导入 `.canvas` 文件
- 保存到思源工作区，也可导出为兼容 Obsidian 的标准 `.canvas` 文件

### 四种卡片类型

| 卡片 | 用途 | 举例 |
|------|------|------|
| **文本卡片** | 写下想法、笔记、备注 | 记录会议要点、写下灵感碎片 |
| **文件卡片** | 关联思源中的文档、图片或其他画布 | 把相关笔记卡片拖到画布上，形成知识地图 |
| **链接卡片** | 放置网页地址 | 参考资料、外部文档的入口 |
| **分组卡片** | 将相关卡片归类分组 | 按主题、阶段或优先级分组 |

- 文本卡片支持 Markdown 渲染，标题、列表、代码块、引用等都能直接预览
- 文件卡片能识别思源文档、块 ID、图片素材和嵌套画布，双击即可跳转到对应内容
- 链接卡片双击即在浏览器中打开
- 从剪贴板粘贴的图片会自动保存为文件卡片

### 连线与关系表达

- 从卡片四边的锚点拖拽即可创建连线，松开时自动吸附到最近的锚点
- 连线支持添加文字标签，标注关系类型（如"导致"、"属于"、"参考"）
- 可设置箭头方向（无箭头、单向、双向）和颜色
- 选中连线后可拖拽端点重新连接到其他卡片，或直接拖到空白处删除

### 排版与布局

选中多个卡片后，浮动工具栏提供 13 种排版操作：

- **对齐** — 左 / 水平居中 / 右 / 上 / 垂直居中 / 下
- **排列** — 横排、纵排、网格（自动计算列数）
- **等距分布** — 水平 / 垂直方向均匀分布
- **拉伸对齐** — 水平 / 垂直方向铺满选区

拖拽分组卡片时，组内所有卡片一起移动。

### 右侧属性面板

选中卡片或连线后，右侧面板可编辑：

- 位置、尺寸、颜色等基础属性
- 文本卡片的内容、文件卡片的路径、链接卡片的 URL
- 连线的起止锚点方向和标签
- 未选中任何内容时，面板显示文件信息、同步状态和最近文件列表

## 快捷键

| 快捷键 | 作用 |
|--------|------|
| `Ctrl+Alt+C` | 打开新画布页签 |
| `Ctrl+S` | 保存当前画布 |
| `Ctrl+A` | 全选卡片 |
| `Delete` / `Backspace` | 删除选中的卡片或连线 |
| `Escape` | 取消编辑 / 关闭弹窗 / 取消选中 |

## 画布操作

| 操作 | 说明 |
|------|------|
| 左键单击卡片 | 选中 |
| `Ctrl` / `Shift` + 单击 | 多选 |
| 拖拽卡片标题栏 | 移动卡片 |
| 拖拽卡片边缘/角落手柄 | 调整大小 |
| 拖拽卡片锚点 | 创建连线 |
| 框选空白区域 | 批量选中范围内的卡片和连线 |
| 右键拖拽画布 | 平移视图 |
| 滚轮缩放 | 以光标位置为中心缩放（0.1x ~ 2.5x） |
| 双击文本卡片 | 进入编辑模式 |
| 双击文件卡片 | 打开对应的文档或画布 |
| 双击链接卡片 | 在浏览器中打开 |

## 设置

| 设置项 | 默认值 | 说明 |
|--------|--------|------|
| 默认画布目录 | `/data/storage/siyuan-canvas` | 新建和保存画布时的基础路径 |
| 最近文件数量 | 8 | 侧边栏显示的最近打开画布数量（1-20） |
| 检测外部修改 | 开启 | 保存时若文件已被外部修改，弹出冲突提示 |

## 使用场景示例

**知识梳理** — 把零散的笔记和资料放到一张画布上，用连线建立关联，用分组按主题归类，形成可视化的知识网络。

**跨工具协作** — 在 Obsidian 中创建的 `.canvas` 文件直接在思源中打开编辑，修改后保存回标准格式，两个工具间无缝流转。

**项目规划** — 用文本卡片列出任务，用连线表示依赖关系，用分组划分阶段，用颜色标记优先级，拖拽排版快速调整布局。

**资料收集** — 从浏览器复制图片直接粘贴到画布，搜索思源中的文档和图片素材添加为卡片，所有参考资料集中在一个画面上。

## 说明

- 底层遵循 `obsidianmd/jsoncanvas` 开放规范
- 保存时会保留规范未定义的额外字段，确保与 Obsidian 的完整互通
- 当前版本优先保证文件格式兼容和核心编辑体验，不追求与 Obsidian Canvas 的完全视觉一致
