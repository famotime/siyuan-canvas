# 演示播放路径录制操作指南

思源画布提供了直观且功能强大的演示播放路径（Presentation Path）录制系统。本文档为您详细介绍如何录制、播放、延长以及截断修改您的演示播放路径。

---

## 📖 背景与简介

**播放路径（Presentation Path）**是您在思源画布中为展示、汇报或整理思路所定制的节点跳转顺序。当配置了播放路径后，画布便能够像 PPT 幻灯片一样，按照您设定好的顺序依次聚焦、放大展示不同的节点。

---

## 🛠 功能界面说明

当进入演示播放模式后，画布底部会出现**播放控制栏**，包含以下按钮：

1. ◀ **上一步（Prev）**：回退到上一个节点。
2. ▶/⏸ **播放/暂停（Play/Pause）**：开始或暂停自动播放。
3. ▶ **下一步（Next）**：前进到下一个节点（若存在单出边，会自动前进；若存在分支则会暂停并高亮出边节点供您选择）。
4. ⏺ **录制（Record）**：
   - **普通状态**：点击开始录制。
   - **带有小蓝点**：表示当前已有保存的播放路径，点击可进行延长或截断录制。
   - **长按（约 700 毫秒）**：清空当前已保存的播放路径。
   - **闪烁红色**：正在录制中，点击可停止并保存。
5. ❌ **退出（Exit）**：退出演示播放模式。

---

## 🏃‍♀️ 操作指南

### 1. 启动演示模式
1. 选中画布中的任意一个节点。
2. 在该节点上方的浮动工具栏中，点击 **播放** 按钮。
3. 系统将根据是否已有保存的播放路径，自动进入以下对应的演示模式：
   - **无保存路径**：直接以选中节点作为起点，进入“图谱自由探索播放模式”。
   - **有保存路径**：无论选择哪个节点，都将默认加载已保存的路径并从第一个节点开始播放。

---

### 2. 录制全新的播放路径
当没有保存的播放路径时：
1. 启动演示模式。
2. 点击控制栏上的 ⏺ **录制** 按钮。按钮将开始呈现红色呼吸灯闪烁。
3. 通过以下方式进行路径导航（它们会被自动记录到播放路径中）：
   - 在画布上双击或单击要前往的相连节点。
   - 在控制栏点击 **下一步**（如果有且只有一个单向连接的下一个节点，会自动前进并记录）。
4. 录制完成后，再次点击 ⏺ **录制** 按钮，系统将保存该播放路径，并结束录制状态。

---

### 3. 延长已保存的播放路径
如果您已经保存了一条播放路径，但想要在其**末尾继续添加新步骤**：
1. 启动演示模式（此时控制栏的 ⏺ 录制按钮右上角会显示一个小蓝点提示已有路径）。
2. 在未选择具体步骤（例如在刚进入演示还未移动，或者播放完全部路径后），直接点击 ⏺ **录制** 按钮。
3. 画布会自动将焦点跳转并聚焦在已保存路径的**最后一个节点**上，并在此基础之上开启录制。
4. 像往常一样，通过点击连接的其他节点以继续导航，新节点将会顺次追加在原有路径末尾。
5. 录制完成后，再次点击 ⏺ 按钮结束录制并保存。

---

### 4. 回退并截断修改录制
如果您发现已保存的路径中，后面的部分节点顺序需要修改，或者想从中间某一步开始**另辟新径**：
1. 启动演示模式，播放您的路径。
2. 使用 ◀ **上一步** 按钮回退，或点击 **播放/暂停** 暂停在您想要修改分支的那个节点（假设原本路径是 `A -> B -> C -> D`，您回退并停留在了 `B` 节点）。
3. 此时点击 ⏺ **录制** 按钮。
4. 系统会智能检测到您当前处于历史步骤 `B` 上，并自动执行**截断**：
   - `B` 节点之后的路径（如 `C -> D`）会被丢弃。
   - 保留 `A -> B` 部分并作为本次录制的初始路径起点。
5. 您现在可以从 `B` 节点点击前往新节点 `E`。
6. 完成后点击 ⏺ **录制** 按钮结束并保存。最终保存的路径将会被成功替换为 `A -> B -> E`。

---

### 5. 清空已保存的路径
如果您想要完全废弃当前已经录制的播放路径：
- 在演示播放状态下，**长按**控制栏的 ⏺ **录制** 按钮（持续约 700 毫秒）。
- 按钮上的小蓝点将会消失，提示播放路径已被成功清空。

---

## ⚠️ 注意事项与已知限制

- **循环路径保护**：在非录制自动播放中，为了防止遇到闭环路径时产生无限死循环，系统会自动过滤掉当前已被访问过的节点。
- **分支自动暂停**：若当前节点有多个出口连接线（分支），为了给汇报人充分的决定空间，自动播放将会在分支处自动暂停，此时只需手动在画布中点击对应的目标节点，或在控制栏点击“下一步”（默认选择第一个分支排序）以继续。
- **自动保存**：录制状态的结束（再次点击录制按钮）会立即将录制完毕的路径完整写入思源的画布数据中，无需额外点击任何保存操作。
